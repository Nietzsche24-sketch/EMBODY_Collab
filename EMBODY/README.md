# EMBODY

🎙️ **EMBODY** is a fully local emotional voice cloning and translation pipeline.

## Modules

-  - YourTTS with Arabic training
-  - Dialect & slang rewriter (Egyptian, Gulf, Levant)
-  - Translation engine
-  - Emotional morphing system
-  - Source and synthesized WAVs
